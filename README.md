# IBI game controller  IBI手柄测试工具


IBI手柄测试工具是一个使用  易语言 编程的手柄测试工具，你可以使用它来测试你的手柄有没有错误

IBI Joystick Test Tool is a joystick test tool programmed in eCode, which you can use to test your joystick for errors.
